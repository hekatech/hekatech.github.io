Win 10 美化小笔记

2023/10/09 字数统计： 1.4k 阅读时间： 2 mins 访问次数： ... 评论统计： ...
  • 维护日志

　　折腾 AviUtl 界面美化的副产品，折腾着折腾着就开始捣鼓整个 Windows 了……

更改 Windows 窗口标题栏颜色

　　众所周知 Win10 的暗色模式有点用处但不多，记事本等诸多应用的标题栏依然是白的，AviUtl 也在白标题栏之列，设置了夜间模式之后看着不可谓不痛苦……
　　活动窗口的改变方法：选一个暗一点的主题色。
　　是的这是唯一的解决方法……
　　设置完毕后发现非活动窗口的标题栏依然是白的，这个时候就需要改注册表，进入 HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\DWM 新建 DWORD（32位）值 并命名为 「AccentColorInactive」，将值改为 FF 开头的 8 位 Hex ARGB 颜色代码。
　　是的这个地方用的是 ARGB 但我尝试使用半透明颜色后他会自动变成 6 位值，也不知道这里用 8 位的意义何在，以后打算搞半透明在这放个新建文件夹是吧 :4o-3:

更改标题栏高度

　　解决了白色标题栏还要解决标题栏过宽把界面拆分得太多的问题……
　　依然是改注册表，进入 HKEY_CURRENT_USER\Control Panel\Desktop\WindowMetrics，修改 BorderWidth 和 PaddedBorderWidth 为 0。

自定义图标

　　尝试用 PS 插件导出了 ico 图标，但不知道为什么缩小的时候会变成龇牙咧嘴的马赛克（就是锯齿很多的意思），多方查询资料后发现 windows 用的 ico 文件需要包括多个尺寸的文件，不然让 windows 自己压缩图片就是会压成这个鬼样子……
　　OK fine。

使用软件：icoFX
虽然是付费软件但似乎免费版也不影响使用，网上随便搜一下也能找到汉化

　　将图片拖入窗口选择 从图像创建 Windows 图标 就会自动生成多个尺寸的图像，然后直接保存为 ico 即可。
　　如果图像尺寸小于 256×256，则需要先导入图像再通过菜单的 图像→从图像创建 Windows 图标 生成图标，勾选 256 尺寸后会自动把图片放大，虽然效果比较模糊但是这个尺寸的图标其实不太用得到，反正我是无所谓【。在意的话可以用别的软件预处理一下图像。

创建图标库

　　在菜单中选择 工具→图标资源编辑器 可以添加多个 ico 文件创建 icl 格式的图标库，即一个文件内包含多个 ico 图标。但是这个格式在 windows 文件浏览器中是不会直接显示图的，我觉得发布少量图标的时候用多个单图像 ico 文件打包的形式也不错。

去除快捷方式箭头

　　还是改注册表！
　　进入 HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Shell Icons，如果没有 Shell Icons 这一项就新建项，然后新建 字符串值 命名为 29，将值改为 C:\WINDOWS\system32\imageres.dll,197 即可。
　　这个操作是修改了快捷方式的角标 ico 文件, imageres.dll,197 是一个接近全透明的图标，详情请【参考此处】
　　当然我自己画了一个更接近全透明的图标，经测试不会自动生成为全黑缩略图，有需要可以【点我下载】
　　下载后将注册表值改为透明图标的路径即可。
