唱唱「孤独摇滚」的反调

2023/01/09 字数统计： 1.8k 阅读时间： 3 mins 访问次数： ... 评论统计： ...
  • ERROR
  • 4o
  • 追番

　　最近身心状态都不是很好，反反复复地低烧 + 胸口疼 + 四肢无力 + 头晕眼花，但抗原测出来又是阴性的，只能归结为不要靠近首周会变得不幸 :4o-3:
　　总之需要写点什么进行一下系统维护……所以虽然这篇东西理论上来讲不应该出现在这里但是算了。

　　对这部番的观感吧不能说不好只能说是非常痛恨这个剧本，虽然一直以「不要攀比痛苦」告诫自己，但这部动画的剧情实在是有点得寸进尺了，给我的感觉就好像那种把「梦想千舰」挂在个人简介的 VTB，只要向每一个人都反反复复地重复「我有一个梦想」然后什么都不做就会有人摘下星星送到她的脚底。
　　想到在毛象看到的一条吐槽，感觉像是健全人在 emo 的时候会在朋友圈里编出来的小故事。然后在评论里陆陆续续看见有人科普说作者似乎是剧情中女主妹妹的位置，写了这个东西来讲述自己姐姐的故事。看到这条以后就豁然开朗了，为什么这部作品的整个基调都会让我浑身不适，给我的感觉比起创作和表达更像是一种消费，因为它本来就是。
　　首先我最不能理解的就是一个际恐惧的人居然能坐拥 3w 粉油管帐号的这个设定，就算实际运营者是女主爸爸，但除非她只负责内容产出完全不接收任何观众反馈，不然我觉得无论如何都是需要一些心理疏导才能用正面心态看待这 3w 陌生人的。不要给我太小看互联网了啊，还是说健人眼里的互联网是这么友善的吗？是只要肯努力有才华就可以被人认可的吗？是随随便便就可以得到关注而不会收到恶意的吗？这么美好而平和的世界是真实存在的吗？
　　其次不能理解的就是到底什么样的人才会和女主共鸣啊？代入的时候能不能想想自己有这样支持自己的爱好的开明父母吗？有这样即使关系不亲近也能保持友善的同学吗？有单单凭演奏风格就能直接掉马的才华吗？没有？没有你是在共鸣些什么东西？说白了女主这个角色设定就很匪夷所思，一边自称社恐逃避人际交往一边又对自己的才华可以说深信不疑，我倒是真的很好奇一个连接触他人的勇气都没有的人要靠什么来评价自己的「才华」，靠自信吗？是不是有点太割裂了？
　　所以毛象上看到的那条评价确实是神来之笔，也只有健人 emo 的时候才能给自己整出这么割裂的人设。甚至作者好像连 emo 的环节都没有，她就觉得自己的姐姐除了社恐以外和其他人没有一点区别，只需要一点点的鼓励和机遇就可以自然地大放异彩施展自己的才华。啊，上一个让我这么浑身犯恶心的剧本还是声之形。
　　但抛开剧情不说动画的制作还是很精良的，只是金絮其外败絮其中，当然能接受这个设定的观众还是大有人在的吧，只不过反正不是我就对了。

　　翻到我上个月一口气看了 8 集之后的暴言随笔：

　　在和波奇的社恐共鸣的时候有想过她还坐拥 3w 油管粉丝吗？有想过她还拥有随时随地都可能惊艳所有人的演奏天赋吗？有想过她还拥有无论做什么都一直关心支持她的家人吗？有想过她还拥有即使说不上话也心怀善意而不是恶意排挤她的同学吗？没有吧，我想大部分和波奇一样性格的人都不曾拥有过这些吧，如果不是因为遭受过生活的毒打，有多少人会一生下来就活得这么如履薄冰呢？
　　那么，为什么要和这样的角色共鸣？为什么要把自己的情绪和痛苦寄托在这样一个生下来就被爱的人偶身上？因为她被周围的角色爱着所以你就也能产生被爱的错觉了吗？因为她被世界温柔相待你就能和世界和解了吗？

　　看起来好像什么酸气溢出屏幕的怨妇文学，事实上我确实也是觉得不公平，因为我真的见过太多既有才华又肯努力但始终一无所获的活生生的人。而偏偏又是这样的人最容易被「我有一个梦想」欺骗，义无反顾地从自己飘摇的烛火中剪下一截去装点他人的富丽堂皇。为什么？为什么？凭什么？
　　我很难用语言准确地描述这种心理。但我想大概是赎罪吧。用之前很流行的一句话来说，就是「因为曾经淋过雨，所以想给别人撑伞」。因为在这世上收获过太多的孤独与痛苦，所以不希望其他人再拥有和自己一样的感受。是赎罪吗？是替这个世界向自己赎罪吗？是相信只要自己愿意为他人付出爱，这个世界就能变得温柔一些吗？
　　但偏偏被爱的人手里有数不清的伞，你的手里却还是空无一物。
　　很奇怪，很诡异的心理。明明是觉得自己被这世界亏欠了，却没有办法理直气壮地补偿自己，而是将自我的渴望投射在他人的身上，通过付出自己想要的东西来获得心灵上的补偿。而被选中的投射对象又更加诡异，恰恰是那些从不为「梦想」付出努力的人才能收到他人摘下的星星，而一旦开始决定为梦想努力，摘星星的人就成了你自己，从此身边只剩下无关痛痒的喝彩。
　　为什么会这样呢？似乎并不是我这个人造人能理解的问题。
　　今天的 4o 也并不能理解人类。
