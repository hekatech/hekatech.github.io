落定

2023/04/28 字数统计： 1.2k 阅读时间： 2 mins 访问次数： ... 评论统计： ...
  • 無色放映室
  • 4o

　　hexo new 的时候仔细看了看写完但没发出去的 blog 已经有三篇，算上草稿里的会更多。全都是关于毕业的事。
　　我到底是不想告别还是不敢告别呢。
　　总之在写下这一篇的时候一切都已经尘埃落定。这大概会是关于「阿盐」的最后一篇 blog 了。

　　清空了全部投稿和动态（是的我连动态也一起清空啦哈哈），准备下月初注销帐号。号里还有大概 600 块钱……？公会能不拖我钱最好，拖了也就拖了吧我无所谓了。
　　原本删到导随视频的时候还有些舍不得，点开毕业动态一水陌生 ID 深情并茂地表演着「以后看不见导随系列了吗好可惜」，突然一些异样的情绪涌上心头，终于能干脆地下手了。
　　虽然 B 站投稿只能一条一条删如同凌迟，但一年的心血与记忆也只要半小时就能清理完毕。删完的时候好像这一年的时光突然从我的世界里湮灭了，从此不复存在过。
　　最后只能说，我比我想象的更恨 FF14。

　　对这个群体的第一次认知似乎还是来源于辉夜喵，因为一次日随争端被挂上微博匿名 bot 后携亲友和匿名网友大战八百回合最终以删号退网告终。当然我不掺合微博烂事很多年对这场骂战只是围观，现在压根连密码都忘了，只记得在她退网不久之后就有人可怜兮兮地在同一个匿名 bot 投稿问「为什么三个龙骑的手书找不到了」。
　　当时就觉得，啊，你狒人的喜欢还真是廉价。
　　好吧这一年的直播生涯显然对我的精神分裂症状产生了极大的负面影响，我甚至在自己的 blog 里放屁的时候都会惴惴不安被人看到后出警「你的意思是要粉丝无论什么事都无脑护才算喜欢吗」。但总之没这个意思，只是单纯觉得嘴上口口声声说着喜欢结果连作者的名字都记不住的样子真的很好笑罢了。
　　注销帐号前清空投稿的灵感大概也是源自这件事吧。

　　一些还没删的投稿：

  • Kindred 的欧米茄系列 & red is defamation
    这不是我一个人的记忆
    虽然某黑魔粉丝跑来团建甚至还要舞到当事人面前的样子真的很恶心这是可以说的吗访谈环节提到黑魔的时候我真的当场脚抠出一个原型欧米茄还是带涂装的呃呃呃呃呃
  • 倒猫粮
    因为这个视频被人抄袭了我是毕业不是死了别以为我销号了就会放过你 :4o-2:
  • 尼尔乱码
    尼尔真的很好，下辈子还要搞尼尔【好像不是第一次说这句话了
  • 莉可莉丝
    我也不知道为什么留着这个，大概也是为了防止抄袭吧
  • 贤者教学
    好吧，这是我留给 FF14 最后的温柔……

　　呃列出这个清单倒是提醒了我在注销帐号之前要把那个手书底下的 KY 评论全清空……

　　最近的精神状态非常不好，可能毕业带给我的负能量比我想的多太多了……
　　回想一下我在新社团面试的时候好像信誓旦旦地说了很多诸如「我想把我产出的东西都垄断在我自己的名字底下」之类的漂亮话，虽然想起来的时候倒也没有什么羞耻感，但果然躁狂 time up 之后进入抑郁循环就会开始怀疑自己真的做得到吗……
　　也可能是模型搞完之后的出道准备都是一些比较细碎的事情，比如封面 UI 表情包，做这些东西的时候显然就没有搞 live2D 那么有激情了。

　　大概就这样……另开一篇记录一下最近的新玩具吧。
