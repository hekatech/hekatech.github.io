另外建议双 T 开怪时开启疾跑一起去 1 点就位，以防止没分摊到暴毙的情况

    • 引导火分摊的人需要站在格子的四角外侧

    • 当紫圈 :p8s-mag: 点 2DPS 时

      DPS 按【内←1234→外】的顺位，在右上角处理
    • 当紫圈 :p8s-mag: 点 1T1N 时
      DPS 按基础站位

第一轮踩塔

  • 2:06 魔法陣起動
    • 第一轮大圈判定后，短毒三人头上会出现「完成概念」
    • 同时场地上出现两个同色塔，塔的颜色是绿 / 蓝 / 紫三选一，需要两名携带「完成概念」的玩家贴贴合成出对应颜色的召唤兽 :p8s-gre: :p8s-blu: :p8s-pur: 来处理

      塔与妖灵王水圈类似，在紫球倒计时结束后才会判定，判定时如果没有人踩会直接炸团
      如果没有合成召唤兽就去踩塔会获得 2w1 一跳的 dot，基本离死不远了

　　此处插播三个小抄宏：

/p 绿 = A + B <se.1>
　　/p 绿 = A + B <se.1>
　　　　/p 绿 = A + B <se.1>

/p 蓝 = A + C <se.1>
　　/p 蓝 = A + C <se.1>
　　　　/p 蓝 = A + C <se.1>

/p 紫 = B + C <se.1>
　　/p 紫 = B + C <se.1>
　　　　/p 紫 = B + C <se.1>

　　即：绿塔由 AB 点的两个概念合成并踩塔，蓝塔由 AC 踩，紫塔 BC 踩，此外建议把宏图标设置成相应颜色的水晶。只踩塔的话其实看头顶图标的颜色就可以，但这个宏可以帮助继续处理后续的机制。

  • 2:13 Conception / 概念合成
    • 大圈判定后首先在场中集合治疗，为了方便合成请不参与合成的毒稍微远离一步
    • 合成后按照 A-B-C 点的顺位各自往南北走踩塔
      如踩蓝塔为 AC 点的毒合成，则 A 点来的毒去北塔，C 点来的毒去南塔
  • 2:18 アッシュブレイズ / 灰烬焰息
    • 在合成过程中 boss 一侧手也会发光，在踩塔的同时判定左右刀，需要在安全半场踩塔
    • 同时人群也要暂时前往安全半场躲避，在左右刀判定结束后再前往第二轮站位
  • 2:19 魔陣発動
    • 塔判定，伤害约 5w

P3：万象灰烬

    • AOE dot 减伤.jpg
    • 做好减伤和治疗后即可前往场边为万象灰烬站位
  • 3:05 万象灰燼
    • 先看分散站位，如图：
    • 接下来是机制解说：
      • boss 会首先点随机 1TN / 1DPS 释放一个半径约半格的爆炸 AOE，附带 9s 火耐降低 :fire-down:，点名共 4 轮
      • 火耐 :fire-down: 剩余约 2s 时会在该玩家脚下生成半径约 3/4 格的黄圈，满血吃不会死，但是会获得这个 :damage-down:
      • 同时场地前三行会出现左右各 4 座塔，需要处理完以上两轮机制的玩家负责踩塔
    • 所以每个人需要做的事就是：分散数自己是第几轮爆炸→放置黄圈→踩塔
    • 塔出现的规律是 1 / 3 塔必定出现在前两行，2 / 4 塔必定出现在后两行
      所以我们规定第 1 / 3 轮被点爆炸的玩家去前排的十字交叉点（就是图上画的位置）放黄圈，2 / 4 在后排
      负责的塔可以提前数，也可以在放完黄圈后找自己相邻四格里快要炸的塔

      如图，黄圈出现后周围四格里低于这个高度的塔必定是你要踩的
    • 由从爆炸结束到黄圈判定约有 7s 的时间，因为将黄圈集合放在两个点，所以左右两侧和场中的直线都是可以站的，请合理规划路线
      示范一种比较麻烦的情况即 H2 / H1 / ST 分别是 2 / 3 / 4 塔，则 H1 和 ST 可以在第四次火点名爆炸后交换位置，等一二轮的黄圈消失后再继续处理机制

P5：概念支配（二运）

在二运所有踩塔机制结束后，boss 会读条「自我概念崩坏」对我方全体释放秒杀伤害
需要在处理完踩塔机制后，使用合成出的风神与火神二次合成出不死鸟来处理

  • 5:33 概念支配
    • 伤害超过 10w 的转场大 AOE，请安排好减伤
    • AOE 结束后会赋予 6 名玩家以下 debuff：
      • 长 :p8s-A: :p8s-B: :p8s-Y:：27s 长「未完成概念」，简称长毒
      • 短 :p8s-A: :p8s-B: :p8s-Y:：7s 短「未完成概念」，简称短毒
      • 单人分摊【？：:p8s-1: 需要单人处理，必定点长毒中的一人
      • 双人分摊：:p8s-2: 必定点长毒中的一人
    • 另外还有两名无 buff 玩家

第一轮站位 & 踩塔

  • 二运第一轮站位如图：

    短 :p8s-A: :p8s-B: :p8s-Y: 分别去 ABC 点，单人分摊在 D 上，双人分摊与另一个长毒在 D 下
    无 buff 的二人去 A 点附近吃大圈获得「完成概念 α」:p8s-aa: 准备合成火神 :p8s-red:
  • 5:40 魔法陣起動
    • 塔出现，与一运同样处理
    • 这一轮的塔必定是蓝色 :p8s-blu: 或紫色 :p8s-pur:
  • 5:41 概念変異
    • 短毒大圈判定，约 7w 伤害
  • 5:46 Conception / 概念合成
    • 大圈判定后回场中吃奶，短毒组按一运合成踩塔
      无 buff 组在 3 点处合成火神 :p8s-red:

      需要注意的是合成火神后会有 27s 的「概念拒绝」:p8s-rej: debuff，如合成得太晚会影响后续的不死鸟合成
      建议在大圈判定后的「概念拒绝」:p8s-rej: debuff 消失后尽快合成火神 :p8s-red:，同时请奶妈多关照一下两位火神玩家

第二轮站位 & 踩塔

  • 二运第二轮站位如图：

    两名火神 :p8s-red: 在 D 点右侧预站位
    踩塔的两个合成兽（ :p8s-blu: 或 :p8s-pur: ）按塔的位置在 D 点上下预站位
    长 :p8s-A: :p8s-B: :p8s-Y: 分别去 ABC 点，第一轮未踩塔的短毒回自己的点补位
  • 5:59 概念変異 + 魔法陣起動
    • 长毒大圈判定，场上出现四个塔
      这一轮的塔是呈四角形状排列的，同时必定有两个绿色塔
      合成顺位也与一运相同

      即长 :p8s-A: :p8s-B: 固定去北，短 :p8s-A: :p8s-B: 与长 :p8s-Y: 固定去南
    • 背不下来也可以利用小抄宏：
      • 小抄宏是 蓝 = A + C，则 A 点来的长 :p8s-A: 去北，C 点来的长 :p8s-Y: 去南；宏里没写的长 :p8s-B: 去北，短 :p8s-B: 去南
      • 小抄宏是 紫 = B + C，则 B 点来的长 :p8s-B: 去北，C 点来的长 :p8s-Y: 去南；第一轮宏里没写的长 :p8s-A: 去北，短 :p8s-A: 去南
    • 但这一轮塔的排列是随机的
      所以在合成时需要用预站位来解决同色塔的顺位
  • 6:11 魔陣発動

  • 6:12 メギドフレイム / 米吉多烈焰
    • 在场中合成的同时场外 ABCD 正点会出现 4 个分身
      一段时间后分身会连线距离最近的玩家，连线消失后会向玩家所在的方向释放直线 AOE
      • 我们规定由 D 点附近的四个玩家按预站位去引导分身
        即北塔去 A，南塔去 C，靠右的火神 :p8s-red: 去 B，靠左的火神 :p8s-red: 在原地
        等待小怪连线后顺时针移动一个标点引导
      • 为了方便治疗抬血，请引导的玩家尽可能靠近场中，但不要踩进场中的四个格子里，大概站在如图的位置引导即可：

        同时踩塔的四名玩家也尽量在靠近场中的位置踩塔，以免吃到场外的直线 AOE

合成不死鸟

  • 6:17 Conception / 概念合成
    • 正确处理二运所有合成踩塔机制后，场上应当还剩下两个火神 :p8s-red:，两个风神 :p8s-gre:，以及两个其它颜色的合成兽
      同时火神玩家身上的「概念拒绝」:p8s-rej: debuff 应该正好在踩塔判定后不久消失
    • 我们规定左 / 上塔的风神 :p8s-gre: 与 A 点的火神 :p8s-red: 合成，右 / 下塔的风神 :p8s-gre: 与 C 点的火神 :p8s-red: 合成
    • 在合成的同时可以放掉近战的三段 LB，同时请治疗务必确保全员满血
  • 6:22 Everburn / 概念生成：不死鸟
    • 风神和火神合成后场上会出现四个不死鸟羽毛图标 :p8s-fur:

      羽毛继续合成后会赋予全员头顶不死鸟图标，同时全员强制 1 血并获得不死鸟 buff :p8s-phn:

      未满血的玩家将会在不死鸟出现的同时变成死鸟被秒杀
      不死鸟会在后续阶段全程给予我方双倍输出的 buff :p8s-2x:
      因此在这里死一个人 = 三个黑头，基本上是注定狂暴了
      重申一遍请务必在不死鸟合成前确保全员满血

      需要注意的是这段动画并不是 boss 上天，所以所有的 dot 依然是有伤害的，不要忘记续

移动塔详解

  • 7:17 移动塔出现
    • 在支配者的一击后首先集合回场中治疗
      boss 会随机赋予 2TN + 2DPS 9s 的土属性耐性降低 debuff :earth-down:
      同时场地中心上会生成四个冒烟的绿圈，随机往 D-A-B 180° 范围的场边直线移动，并会在移动到场边后爆炸，即「移动塔」

      在踩塔后也会获得 9s 的 :earth-down: debuff
      携带 debuff 的玩家无法踩塔，所以需要：
      • 支配者的一击后无 debuff 的四名玩家处理第一轮踩塔
      • 支配者的一击后有 :earth-down: debuff 的四名玩家处理第二轮踩塔
  • 7:24 地盤隆起
    • 第一轮塔判定，踩塔的顺位如下：
      • MT-ST-H1-H2 处理 D-A 的两个塔
        即【MT】固定去左一，【H2】固定去左二
        【ST】与 MT 同组则去左二，不同组则去左一
        【H1】与 H2 同组则去左一，不同组则去左二
      • D1-D2-D3-D4 处理 B-A 的两个塔
        即【D1】固定去右一，【D4】固定去右二
        【D2】与 D1 同组则去右二，不同组则去右一
        【D3】与 D4 同组则去右一，不同组否则去右二
    • 每个人在判断完自己有无 :earth-down: debuff 后
      再看一眼前文提到的关键队友是否与自己的 buff 相同
      就可以判断自己要去哪个塔了
    • 需要注意的是塔的刷新位置较为阴间，在初始移动阶段可能会出现两个塔黏在一起无法分辨的情况

      请冷静判断塔的刷新位置，在塔移动约半格之后再开始移动也是来得及的
  • 7:30 地盤隆起
    • 第二轮塔判定
  • 7:36 アイアンアゴニア / 心孤意怯之惧
