　　之前画的 emoji 虽然对低分屏显示比较友好，但也为此牺牲了一定程度的美观性……时隔大半年重置了一下：:4o-1: :4o-2: :4o-3: :4o-4:
　　算是在可见度和美观性中间选了一个比较折中的画法？

　　←还有一个奇怪的 gif，是说最后还是带病上班了……（没开播，只是画画）
　　虽然理论上应该开播画这个水一下时长，但是人类只会影响我工作的速度！我觉得我的粉丝群体还不足以支撑我开工作音直播，如果能成功从游戏主播转型成游戏创作型主播再考虑工作音吧。
　　当然也不是没画画摆烂过。但果然生病的时候不想再戴耳机回弹幕，好累。
　　画完这个才发现虽然正经插画的水平退步了，但画 Q 版的水平好像有一定的提升，也就是「概括」的能力变强了。也不知道算好事还是坏事。总之还是要多画。
　　啊不行，果然感冒的时候是完全没办法正常组织语言的，我放弃了【

CSS

　　新旧两个版本的 emoji 构图不一致导致 CSS 偏移也需要调整，开始是想着用通配符选择器把旧版 emoji 统一设成旧的偏移，结果不知道为什么实现不了。
　　所以现在是给四个旧 emoji 逐个设置了偏移量。

　　添加新 emoji 的时候突然觉得每画一张新图都要重新写一行 CSS 也太麻烦了……应该用 JS 给每个 span 自动生成背景图的，但是我现在的图片放在伪类里，不知道能不能用 JS 控制……
　　啊不行，脑子已经完全不转了，等感冒好了再说吧……希望明天能顺利开播 :4o-6:
