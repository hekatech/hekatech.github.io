　　当你看到这篇东西的时候……不出意外我大概是终于克服了我的解离障碍把 blog 公开了。（虽然可能是暂时的）
　　之前一直遮遮掩掩地不肯关联直播账号和 blog，但随着直播在日常生活的比重里越来越大，实质上这个 blog 已经变成了《虚拟主播从入门到开摆》的心路历程。甚至我还做了站点地图，本身也有粉丝搜索找到这个 blog 的前例，再切割实在是已经没什么意义了。
　　正好我也希望修正一下自己温驯的形象，说得漂亮一点是「希望我能通过才华获得你的认同」，不过我的真实想法果然还是「人类什么的去死算啦！」

　　真的很喜欢一些同人女，能不能看懂她们刷的东西不重要，主要是喜欢这种与世无争地发疯的精神状态。（同人警察爪巴）
　　虽然这话看起来好像有点冒犯但我是真的喜欢，所以决定当单推我自己的同人女，哈哈！

设定更新

HKT-924

　　将 HKT-63 修正为 HKT-924，为「くちなし」的谐音。
　　呃我是不会告诉你我手动挨个查找替换了所有 post tag 的
　　本站实际运营者，负责时空观测系统「無色」Achrom-4o 人格数据的设计、备份与安装的随行支援系统，在漫长的旅行中已经丧失了对人类的耐心，决定冷眼旁观人类的灭亡。
　　常见外观为灰白色球形悬浮物，但搭载了名为「人工天使 Suzushi = 涼栀」的投影程序，偶尔也会以白色长发的少女形态出现在人类面前。没想到吧这个球可能也准备出道了
　　下辈子的人设已经写好了哦，就算毕业也没什么可怕的了.jpg

Achrom-4o type α

　　HKT-924 为了筹措活动经费制作的虚拟主播 1 号机。
　　由于未为人格数据装载保护程序，在运行 6 个月后就不幸过载，已回收并进入休眠状态。

Achrom-4o type β

　　HKT-924 制作的「無色」Achrom-4o 2 号机。
　　与 1 号机不同，是正式服役的「無色」机体，目前在地球进行短期休假打喷中，顺便也暂时接替了 1 号机直播的工作。
　　吸取 1 号机的耐久问题教训，装载了 HKT-924 同款的人格数据保护程序，但该程序对于 1 号机的观众来说似乎有些过于激进了，对此 HKT-924 表示：「爱看不看。」

Achrom-4o accord

　　设定上没有来过 21 世纪的地球所以非常幸运地不会出现在直播中的「無色」系列原型机。是实际在干活的那一位。
　　编号来源于龙背系列的雅科尔。
　　与以上两位「4o」共用一个署名 tag，主要负责文书记录工作。或许在未来会非常偶尔地会在视频或特殊企划中客串一下，为人类讲述一些不属于这个时代的知识。

HKT-α5

　　似乎是 HKT-924 在支援「無色」系列时空观测系统活动途中遇到的一只橘猫。
　　因为深受所有人造人的喜爱，被数据化后上传到了 HKT-924 的服务器中。
　　由于未知错误与 HKT-924 的系统融合成为了人工智能猫猫，短暂地接替了两个月 1 号机的任务后成为了直播间的吉祥物。

Gitlab 从入门到放弃

　　我最开始折腾 gitlab 的需求：

  • 【紧急不重要】另搭一个 blog 用来存放乐谱、绘画等公开素材
  • 【重要不紧急】从私有库外链 CSS 文件和图片素材为乙方提供一键式直播间装修

　　实际上达成了 1 就也可以完成 2 了，然而花了一晚上才堪堪把 hexo 源文件 push 到 gitlab 上的我实在是没有耐心再搭什么 runner。甚至连笔记我都懒得排版了！！！
　　再加上仅有的这个休息的周末又被感冒占据了 90%，前后合计了一下还是决定暂时放弃了并且直接开摆公开了我的 blog，私有库外链这个东西等我真的约到 CSS 稿再说。
　　但在折腾的过程中我也不是毫无收获的，至少我现在已经熟练掌握了用 git 往仓库传文件的流程以及本地 ssh 文件管理的方式，嗯……至少！至少公开 CSS 的配布不会有任何问题啦！

　　当然最主要的放弃原因是 github 可以给每个仓库设置单独的域名了，我可以先短暂地公开一下 blog 见状不对就直接搬东西跑路，跑路也不会影响已有仓库的外联，好，太好了。
　　不过 blog 公开以后应该是大概率不会再取消了？毕竟你要是想用这个站里的内容攻击我首先得做出比我还好的东西才行吧，不然看起来是不是太可怜了？是个人都会觉得你在无能狂怒吧 :4o-7:
　　讨厌人类。真的因为 blog 里的内容出现什么炎上事件我也会选择毕业而不是关站，毕竟直播可能会是暂时的，但是 blog 是要写一辈子的，为了一时的工作放弃我一辈子的爱好太划不来了。
　　不过也大概是杞人忧天的成分多一点吧？哪有那么多对我本人感兴趣试图了解我的观众啊
