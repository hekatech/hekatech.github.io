　　虽然医生叮嘱的饮食规律作息规律控制咖啡因摄入我一条都做不到就是了，我要是做得到我还至于到你这来吗.jpg

  • 舰长礼物

  • 发了忏悔录的翻唱

  • 把绝龙诗打到了 P4【！？
  • 做了两个版本的弹幕样式

弹幕样式笔记

　　虽然我开这篇文章的初衷就是为了这个，但我现在困得头盖骨上有小人跳绳所以决定还是睡醒再写吧……！

　　一开始直播间用的弹幕姬是 bilibili livechat，在直播的间隙写了一下弹幕样式，结果还没用上几天就被 B 站屏蔽了……给作者发了 issue 修复得倒是挺快，然而在他连夜修 bug 的同时我也连夜给 Blivechat 又写了一份样式……
　　那么以下是两个弹幕姬的使用体验对比：

  • BBLC
    • 轻量简洁，自带独立的 CSS 文件，写样式也很方便
    • 缺点是礼物和 SC 无法高亮，也无法显示 UP 主大表情，算是轻量的代价吧
    • 以及「从顶部开始加载弹幕」的功能和我设想的不太一样，另外似乎是通过弹幕和空 block 的接触判定消除弹幕的，用 flex-derection: column-reverse 调整弹幕进入方向之后弹幕无上限加载
    • 我感觉这也是一个 issue 就能解决的问题，你给弹幕设一个加载上限就好了嘛搞什么碰撞验证……
  • BLC
    • 功能比较多，可以单独设定上舰和礼物特效，也可以显示大表情
    • 缺点 1：CSS 样式和动画随机分布在 dom 内联和自带 CSS 文件内 :4o-4:
    • 缺点 2：公开用于示范的 CSS 样式内每一行都后缀了 !important 并且样式和选择器的语法都毫无规律，我家猫写的代码都比你这玩意好维护 :4o-2:
    • 缺点 3：神™弹幕载入动画不是 CSS 的高度渐变而是用 JS 控制整个窗口抖一下，谁教你这么写的？
    • 因为自带弹幕上限设定可以通过修改 flex 排序满足从顶部载入弹幕的需求，但是由于上一条提到的动画存在，导致视觉效果上弹幕依然是从下方加载的，到底是谁教你这么写的？？？
    • 还有最无语的一条是不管我怎么设置它都不显示免费礼物，什么 big big V 结晶模拟器

　　从代码审美的角度出发我肯定是更倾向前者，倒不如说实际上手体验了一下之后我对后者的生态环境更加丰富这一点简直难以置信【但我同时也是个要吃饭的 V，客观来讲高亮礼物和 SC 确实是有促进消费的作用……:4o-6:
　　总之自己的直播间应该暂时改用后者了，不过如果以后开放弹幕样式定制业务一定会给乙方强推前者【

　　弹幕样式是直播写的，一边写代码一边痛骂了 !important 和 ow2，也没忍住在直播过程里放出了 blog 的页面。不过没有公开地址，因为「这个博客里写了很多阴暗的东西，还没有做好公开的准备」。
　　所以这一篇的文件名是「show off」。我好像就是不太能藏得住东西的人，新做的模型也好写的小说也好只要做了就几乎一定会晒出来，只是时间早晚的问题，也因此「身份隔离」好像成了和「好好吃饭」一样困难的事。
　　但是人类为什么要这样躲躲藏藏地活着呢，我也不明白。

龙诗和一些不良情绪

　　原本的九月计划：晚上 8 点国服巴哈，12 点以后国际服零式。
　　结果刚好国服开龙诗 + 国际服开零式，巴哈队迟迟组不到人，国际服的零式也是从找队开始就坎坷不断，一直打到 10.8 才过本，为此龙诗还推迟了一周开打。

　　打完这个零式整个人就是很累，非常萎靡。具体发生了啥事不想再记了，总之得到的教训是三绝四绝和奶妈的开荒以及排轴能力毫不相关，下次绝对不会再报多余的期待也不会给多余的脸色了。
　　……回头一看我这个头一次打高难的 B 学者不也打到 P4 了，可见绝龙诗也就不过如此吧【
　　因为零式带来的负面情绪和匪夷所思的作息导致龙诗开荒也迟迟没有状态，讲实话我一直到现在都没抄完奶轴哈哈，刚开打那几天还被队里的过本选手当作重点关照对象，一进 P2 就盯着我的位置开始给我报塔，结果是我更紧张了，差点直接自闭【但总之还是友好沟通了一下顺利解决了这个问题你别报了我听见语音里有人点我名更慌了啊啊啊，现在已经能游刃有余地复活营救看陨石一条龙了！

　　当然龙诗压力还是不小的，虽然主要的压力来自队友进度不一，根据已知信息推测大概是 2 过本 + 1 P4 + 5 从 0？整个队应该是占星亲友拉亲友组起来的，语音里很热闹，但我直到最近几天才能堪堪分清是谁在说话……打绝神兵留下的亲友团 PTSD 还没治好，外加学者的熟练度不够，前期准备做得并不充分，以及同时还要忙直播活动和新衣制作，确实有一种压力只能转化为病例的感觉……但队友人都挺好的，只是我自己没办法再信任人类了【
　　和打绝亚的时候我本身就是二运复建不一样，我进度比队友快的时候我可以非常积极地活跃气氛优化解法复盘做作业，但是反过来我进度落后于别人的时候我是万万拉不下脸拜托队友做这些事的，倒不如说队友这么做了我反而会有一种迷之负罪感……也可能是一起打过绝亚的占星和龙骑都不怎么开麦，我一个社恐在陌生环境里绝赞应激 :4o-3:
　　写出来了以后感觉好了不少，总之 11 月至少不用再忙活直播活动了，努力调整状态好好打本吧。

生日会活动

　　先草率地写一点筹备过程，详细的在舰长礼物那篇继续吧。
　　活动没整出来啥活，也不是很多人看，只是普通地开了歌回，而且椅子太高了坐着驼背嗓子状态也不是很好。但是久违地和大家聊了很久的天，还是喜欢这样的感觉……宁可粉丝少一点，和每个人都更亲近一点。
　　虽然提问箱一直到最后一天才有人投稿（还有播一半才来投的！），不过还是收到不少问题，也讲了很多很多……决定之后作为常驻环节吧，可以不定期录一下视频回复。

粉丝群 & 舰长群

　　生日会直播时最无语的突发事故：舰长群被屏蔽了。虽然有人说要切片要切片感觉切出来可能也确实有一些节目效果，但是我觉得好恶心啊……
　　讲实话吧我其实挺火大的，加上前几天有人在群里发了子言的 R18 MMD 还有一群人热烈讨论这种情绪就更加激烈。总之我是当场开骂了:「这么喜欢 GHS 为什么不去相亲相爱一家人群里发，是因为没有家人吗？」:4o-2:
　　总之换了新的舰长群，安慰自己至少有一段时间不用面对「主播的舰长群都快 100 人了，想必很快就能百舰了吧」这种尖锐的问题了，但是心情还是很难平复。事情过去很多天我都觉得这个行为很匪夷所思，传播 V 的 R18 内容，在 V 的舰长群里发别的女人，在 V 的舰长群里发黄图，无论哪一条单拎出来都足够直接判死刑了，怎么还有人能同时完成这三项壮举的？
　　我一直觉得当代互联网的基本礼仪是不能发给你妈看的东西也就不要随便发给陌生网友，只能说有人的户口本可能不是很完整吧。
　　粉丝群因为直播机器人反复被举报去世干脆摆烂了，解除了禁言改成手动通知。
　　就……自我安慰一下，也算是象征了一个新的开始……吧？
