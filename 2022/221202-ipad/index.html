iPad 相关配件 / APP 测评

2022/12/02 字数统计： 3.1k 阅读时间： 5 mins 访问次数： ... 评论统计： ...
  • 维护日志
  • HKT-924

　　虽然这好像和站点维护没什么关系，但人造人的生活当然也是系统维护的一环！所以就放在这个分类里了。
　　啊为什么 6.2 快开了我一不去排龙诗奶轴二不写我的图文攻略反而在这里打字摸鱼呢，为什么呢 :4o-3:
　　总之把 surface 送给母上换来了一台 iPad，以下是近期一段时间的使用感想……

类纸膜 & 笔尖

　　购入 SZYMT 家二件套，总体感觉上没有 surface 那么惊艳了。
　　先说膜，和几年前购入的相比增加了清晰度，随之而来的就是摩擦力也下降了，大概是素描纸和普通书写纸的区别？然后给出的补偿办法是给笔尖附赠了柔韧头，然而这个柔韧头与类纸膜摩擦会发出指甲刮黑板的声音……我即刻告辞。
　　笔尖好吵，真的好吵……从书写舒适度来说其实原装笔尖 + 类纸膜还是最好的，但是原装笔尖在类纸膜上真的会变成粉笔，肉眼可见地掉渣 :4o-3: 这个针管笔尖只能说差强人意吧。
　　另外要吐槽一下的是给我发来的贴膜不知道为什么摄像头的开孔是错的，但是一来我本来就不用前置摄像头二来我真的不想再折腾一次贴膜了，综上所述决定当作无事发生。

文字类 APP

　　首先声明我的需求：

  • 用来写随笔 / 笔记 / 更新博客的文本编辑器
  • 可以多端同步或者导出 markdown 文件
  • 可以显示全角空格
  • 可以生成和显示单行换行符

FlowUs

  • 笔记 ✓
  • 多端同步 ✓
  • 全角空格 ×
  • 换行符 ×

　　山寨的 notion，合计了一下 ipad 的翻墙难度还是毅然选择了山寨版。
　　比原版多了自带最多 5 个用户 + 5 个外部编辑者共计 10 人的协作位，用于固定队填奶轴减伤轴非常合适。虽然腾讯文档的多人协作更加方便但是 excel 做下拉菜单实在是太恶心了………
　　顺带一提这玩意在 iPad 上用着比网页端丝滑多了，难以置信，但是不支持半换行符，应该还是只会用来做奶轴，没办法满足我对文本编辑的审美。

obsidian

  • 笔记 ✓
  • 多端同步 ✓
  • 全角空格 ×
  • 换行符：支持但不算完全支持，单行换行符可以显示但无法显示任何缩进

　　长得好看但是用起来只能算差强人意（把我的全角空格吐出来），具体体现在他会把全角空格识别成和 tab 键一样的缩进符，所以在编辑页面下即使换行后第二行也会缩进同样的距离，在浏览页面下只有双行换行符后的第一行才能显示两个空格。你不如一刀杀了我算了……
　　电脑端是差强人意移动端就只能说稀烂，如果它不是紫色图标我一定会当作没认识它。

metion

  • 博客编辑 ✓
  • 多端同步：仅限 iCloud
  • 全角空格 ✓
  • 换行符 ✓

　　中文排版完美符合我需求的 markown 编辑器，可以导出 md 文档发到电脑单向同步，但是终身买断价 RMB108，呃，打扰了。
　　另外这个 app 的最佳适配场景是外接键盘，但是对我而言绝大多数外接键盘的时间肯定不如直接开电脑，不然搞不好真的会头脑发热 -108……

simplenote

  • 笔记 ✓
  • 多端同步 ✓
  • 全角空格 ✓
  • 换行符 ✓

　　很丝滑的多端同步 + 可以显示全角空格的纯文本编辑器，终于等到你【唱
　　优点是简洁，缺点是有点太简洁了但是真的完美符合我的需求，这篇文章的初稿就是在 iPad 上用它写的。另外移动端的 preview 也不显示半换行符，不过我只用纯文本页面也够了。

幕布

  • 笔记 ✓
  • 多端同步 ✓
  • 全角空格 ×
  • 换行符 ×

　　我知道它是山寨的 dynalist，但是 dynalist 的 app 做得真是一坨……不过这个感觉还是比较适合手机做大纲，pad屏幕太大了不够专注。
　　有了 FlowUs 使用率大概会比较成迷……

joplin

  • 笔记 ✓
  • 多端同步 ✓
  • 全角空格：未测试
  • 换行符：未测试

　　iPad 上没下过但是就凭他电脑端不可自选安装路径占走我 1G C 盘空间就是死刑。

Effie

  • 博客编辑 ✓
  • 多端同步 ✓
  • 全角空格 ✓
  • 换行符 ✓

　　metion 的颜值低配版，比 metion 多一个 PC + 安卓端的同步，然后定价订阅制 168 一年，是不是疯了？

手帐 / 日记 / 小组件类 APP

　　刚买来 iPad 真的有想要做电子手帐，后来发现我这个电子农奴哪来的时间干这么小资的事情……大部分都只简单体验了一下就搁置了。

GoodNotes

　　看起来挺好的电子手帐本，可以自定义 PDF 模板做出各种好看的笔记，如果我真的有时间折腾它就太好了。
　　上半年买的各种本子彩墨印章贴纸都还在闲置中……说起来手帐和博客确实是不太一样的，可以写一些比较私人的阴暗 B 言论，但现在的手帐已经比较接近拼贴画艺术了，不太适合我这种每天连睡觉的时间都不够的猝死区底边游戏主播……
　　大概等打完龙诗和零式的长草期会抽时间研究一下？呃呃打完要劝灯老师先去兵兵给我一点大脑按摩的时间，这都高强度开荒了半年的高难了，机体要撑不住了，放我摸鱼！我要摸鱼！

Days Matter / 倒数日

　　核心功能是倒数计数，可以在你的 iPad 桌面上显示一个硕大的：离国服 6.2 大概还有 18 天！可以说是焦虑症患者必备自虐 app 了。
　　小组件都可以换背景图，也有纯图片组件可以用来装饰首页，绝大部分功能都不收费，比那些专门做桌面美化的 app 良心多了。虽然功能没有专业的桌面美化 app 花样多，但核心的日历时钟和占位符都有，对于我来说挺够用的。

iNote / 灵感笔记

　　一个比较简洁的桌面笔记应用，和上面那位似乎是同一个厂出来的。
　　但我并不用它来做日常笔记，只准备用它来写一些灵光一现可能也比较华而不实的句子或者歌词，确实很符合它的名字，只是记录灵感而已。
　　顺带一提的是内置统计系统，有方格图统计你哪天写了笔记，这个我挺喜欢。但同时它又自带一个看起来非常没有文化的成就系统，统计你已经写了多少字，可以等同于《兰亭序》《出师表》《红楼梦》等等名著，我……呃……只能说还好我应该不太会点开它的内部功能只会在桌面小组件上写东西……

日程类 APP

　　还是先上需求：

  • 以月为单位的待办清单
  • 可以兼顾记录无 ddl 的待办事项
  • 有桌面组件

　　总结：我其实就只是想要一个能写字的电子台历。结果试了一圈居然没一个符合需求的？？只能用系统自带先凑合着。

极简待办

  • 月待办 ×
  • 无 ddl 待办 ✓
  • 桌面组件：有，但是很丑

　　追求极简设计有两种后果，一种是清爽中兼具高级感，另一种是刚刷完墙就进入闲置状态的毛坯房，这个 app 很显然是后者。
　　我不知道这种简洁到连基础 UI 都不保留完全由手势来操作的设计到底是谁先发明的，我只想吐。另外引用一下我对 mastodon 4.0 版本 UI 的评价：「任何左右边距小于 2em 的页面设计对中文来说都是错误的毁灭性的无法阅读的。」此外作为一个在平板上使用的 app，左右的缩进显然还承担着防误触的需求，然而开发者并不在意，我只能说这大概就是典型的程序员审美吧……

我要做计划

  • 月待办 ✓
  • 无 ddl 待办 ✓
  • 桌面组件：有，但是仅支持显示当日范围内的待办

　　设计很可爱的日程清单打卡 app，但是对 ipad 的适配为 0，横屏状态下只能显示一个竖着的方块就算了，竖屏也没法全屏显示我是没想到的……这个特性导致了在上面打字变成了一种折磨。
　　给待办事项按重要 / 紧急分了四个维度，确实比较适合时间管理，但和我的需求不符合。另外桌面小组件只能显示每日的待办内容，最长也只能显示一周内的事件，总之设计上有一些巧思但我用不上，适用对象应该是主要使用手机的学生党，起码和上面的相比还是友好一些……
　　缺点是有很多花里胡哨的签到打卡代币功能，比起生产力软件其实更像一个手游。呃不过想到叮叮倒也很正常……

Microsoft To Do

  • 月待办 ×
  • 无 ddl 待办 ✓
  • 桌面组件 ×

　　在 iPad 上用田牌软件是否搞错了什么【
　　功能和 iPad 自带的备忘大同小异，又不能放桌面还强制要求登录微软帐号，告辞。

　　最后还是放弃了月待办继续手写台历……其实月计划基本都是和直播 / 视频相关需要在电脑前完成的工作，挂在电脑旁边看起来还更方便一些。

　　总之一番装修之后我的 ipad 桌面长成了这样，要用的 app 比较少主要用小组件填充。
　　体感上 iPad 其实不太符合我对文字生产的习惯……当然 procreate 也没好到哪里去就是了，等右下角那张图画完我再单独开一篇文章吐槽它吧，好痛苦，是真的画得我好痛苦……
