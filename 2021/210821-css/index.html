折腾 CSS

  • HKT-924

　　今天的成果全部是 css 伪元素。
　　直觉告诉我这是个不太好维护的东西，但鉴于 1 本站是单人维护不需要交接 2 ejs 你不如把我杀了吧，我还是快乐地敲下了无数个 ::after 和 ::before 。

　　我甚至用伪元素把默认列表的样式都换成了二维码！很赛博！很 suki！而且这个东西看起来真的有好多功能可以挖掘，比如用纯 css 实现在最新文章左上角标一个 NEW！
　　当然这个功能显然不太适合本站，so，日后再说.jpg

　　今日成果： 　　

  • 重置了列表样式 ✓
  • 完成了页面标题样式 ✓
  • 更新了正文链接颜色 ✓
  • 修复了归档按钮动画卡顿的问题 ✓
  • 增加了画廊页面 ✓
  • 添加了行内代码框、引用框和 block 代码框样式 ✓
  • 解决了背景图夏姬八动的问题 ✓
    • 在 body 的 css 样式中加入
      background-attachment: fixed;
      background-size: cover;
      即可

　　待解决：

  • 手机排版惨不忍睹，等我研究一下那个手机端的 title 怎么放出来…… ×
  • 标签和分类页面，还是没做（没做） ○
  • 文内小标题、分割线和表格的样式还没想好 ○

　　新问题：

  • 把浏览器字体设成了 Noto sans CJK，但是渲染效果比微软雅黑更令人疑惑了……
  • 需要一个把弯引号替换成直角引号的功能……加入 #hekation
  • 另外也需要一个在正文输入 tag 的功能，把 '#' + ~ 替换成按钮点击之后弹出搜索框搜索相应关键词……总之我先把这个正文 tag 打在这里，什么时候想起来再实现它吧【你
