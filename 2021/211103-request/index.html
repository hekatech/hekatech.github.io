约稿须知

约稿人需要提供的信息

  • 人物设定：发型发色 / 眼睛的形状 & 颜色 / 其它五官 / 大概的服装风格 / 需要凸显的气质
    需要尽可能详细且具象的文字描述，有参考图是最好的
    • 便于理解的设定描述举例：银色齐刘海短发，略微下垂眼，红色眼睛，脸稍微圆一点，衣服是毛茸茸的卫衣，帽子上加一对兔耳朵？希望是那种可爱的软绵绵的女孩子
    • 令人困惑的设定描述举例：发型清爽一点，眼神慵懒但是有神，衣服我也不太清楚你看着画，气质像兔子一样的那种
  • 联系方式：QQ / 微信 / 毛象私信
    如果留的是微信最好额外再留一个邮箱用来发送成稿大图，因为微信貌似无论怎么发都是会缩图的
  • 其它要求：构图 / 背景 / 整体色调 / 人物姿势等
    如果没有特殊要求默认自由发挥
    但请务必在草稿阶段就确认好具体的造型，之后再修改是非常麻烦的（色调除外）
  • 关于参考图的说明
    • 正确的参考图使用方式：淘宝卖家秀图示意服装 / 姿势，用动漫角色演示发型 / 眼睛形状等
    • 错误的参考图使用方式：直接拿别人的画稿告诉我「请用这种画风画 xxx」或者直接让我照着画一模一样的东西
      如果你喜欢某张图的画风请直接找原作者约稿，「模仿画风」的委托无论对我还是原作者来说都是非常不礼貌的

流程与工期

  • 确认草稿后支付定金
    伪厚涂和平涂的草稿会细化到脸，水彩草稿的草稿（好绕！）只有火柴人和基础构图
  • 支付定金后进入排单队列，绘制需要 2 ~ 7 天
    如有特殊情况 ex 生病 / 加急插队等会另外说明
  • 线稿绘制完成后会再确认一次，确定之后原则上就不再接受造型上的修改了，改也可以，但是要加钱【
  • 绘制完毕后会先发送小图示意，支付尾款之后再发送高清大图
  • 完稿之后默认公开 带水印小图 用于展示，六个月后会公开无水印版本，如有特殊需求（如不希望公开 / 不希望解禁水印）请提前告知

付款说明

  • 本页中所有价格单位均为 RMB
  • 本页中所有价格均为 起步价，最终价格可能视绘制难度和修改次数上浮，一般上浮不超过 50%
    • 加急 / 插队上浮 100% 起
    • 复杂的场景和人物 / 服装设计 另行计费，不在基础上浮范围内
  • 优先付款渠道为微信，也支持支付宝但是尽量走微信
