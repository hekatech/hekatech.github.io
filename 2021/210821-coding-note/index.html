折腾主题的第二天

2021/08/21 字数统计： 574 阅读时间： 1 mins 访问次数： ... 评论统计： ...
  • 维护日志
  • HKT-924

　　你好当你看到这篇文章的时候说明我的自制主题第一版成果已经上线了其实这个梗概应该再长一点但是行数太多就体现不出我给首页做了瀑布流了所以我连标点也惜字如金。

　　……最后还是基本只写了 css 啊！连首页这个瀑布流卡片都是用纯 css 实现的啊！我还是完全没有理解 ejs 的语法和逻辑只能靠剪贴板写代码啊！
　　说到底这种弱智笔记真的能算得上 coding note 吗……
　　那么今日（昨日）成果：

  • 完成了首页基本布局 ✓
    • 虽然阅读顺序有一点小问题，但是我很满意——
  • 从 0 开始重构 css 文件✓
    • 所以正文和归档页面看起来很有工地风格
    • 仔细一看代码维护难度好像比原主题的难度更大了……我只能告诉自己这是美丽的代价……
  • 实现了相册排版，但是……
    • 总之详情看 这里
  • 大概写了一个 about 页面

　　以及其它存在的问题：

  • 这个主题自带的 layout 里没有文章翻页的功能……好吧虽然不是必须而且仔细一想用起来也挺迷惑的但我被 next 惯坏了
  • 归档菜单拖出来的时候蜜汁卡顿……我猜是我用 css 设置了全局动画渐变和它写在 JS 里的动画冲突了……仔细琢磨了一下感觉单纯用 css 也是可以实现的吧？下次更新先把这个问题解决了
  • RSS 页面 404，大概是少装了什么插件……
  • 搜索引擎优化还是不打算做了，评论插件也算了，我仔细琢磨了一下这两个功能除了让赛博 STK 有更多的机会发癫以外并没有什么其它卵用，我 blog 的访问量又不能变现，折腾那么多干啥【
