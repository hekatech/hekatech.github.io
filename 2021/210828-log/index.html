　　理直气壮地：一周没更新是在打 FF14。
　　我可是有三个号的尼尔 CD 要清啊！
　　——截止本篇发出前 9S 的 CD 依然没有清掉，尼尔奶妈为何如此超生……

　　确认了人工智能的人设 球设 以后把日记的标题改成了 log。不是 logs。
　　那么身为一个不需要睡觉的球，我自然是从昨晚通宵到了现在……

　　近日游戏成果：给 9S 黑了黑 pod，话说这玩意是必出的我还开了一人 40w，难怪人一下就满了【过两天给大号也整一个
　　2B roll 了龙骑衣服（没有龙骑）T 装 + 手套，因为五五手套太纤细了和圣龙鞋子不搭还是含泪套娃，这鞋子的模型真的该死的完美但它就是该死地长毛！
　　大号 roll 了奶法衣服外加……嗯？这周是不是第三周了，我这周 roll 了啥？
　　而且尼尔周常垃圾没捡。上周就没捡，这周已经星期六了……我觉得我可能又要忘了……

　　新建分类「ERROR」用来放摸鱼日记，好！
　　其实本来今天是想把分类页面的 layout 写一下的，但是西柚子个站的分类界面排版就一直不太对，于是突然有了手动写的想法……
　　主要这个站还是侧重「展示」而不是「记录」，像这样碎碎念的内容应该是极少数，先写个大纲再往里填内容或许会比较好？

　　前几天帮塔塔解决了一些 hugo 发布的问题，看到 hugo 的渲染成果和简洁易懂的 layout 文件不得不说有亿点心动，但是既然 hexo 已经搞到一半了，只能说下次一定。
　　另外打算把浏览器标题做成 「页面」- 站点名 的形式，显然这个功能是要改 layout 的，位置我也找到了，但是我不知道从何下手……总之有空去找 next 抄一抄。

　　不错最后还是把分类页面大概写了一下，并且用一行代码解决了分类页会渲染多余的日期的问题。说起来我倒是想在每个分类的子页面里放一个最后更新时间，不知道能不能实现呢……
　　另外把之前打的所有弯引号修正成了方引号，不知道为什么就是觉得方引号会更有 AI 感？可能是主流「科技感」设计都是追求扁平化，使用无衬线字体比较多，但是弯引号一般出现在衬线字体里吧……

　　那么目前除了搜索和 RSS 两个需要插件完成的功能之外，「無色人間」已经大体构建完毕，对 ejs 的理解没有丝毫进步，css 倒是莫名精进了不少……
　　人类果然很难跳出自己的舒适区，球也是如此呢【。
