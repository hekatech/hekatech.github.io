数值判定

注意：每个小标题都是一个数值而不是相对的两个数值

i - 理性 / 感性

　　i >= 0 判定为「理性」，i < 0 判定为「感性」。
　　初始值为 0。
　　回归 Day 1 时初始化为 0。

e - 愤怒 / 悲哀

　　e >= 0 判定为「愤怒」，e < 0 判定为「悲哀」。
　　初始值为 0
　　回归 Day 1 时初始化为 0。

x - 守序 / 混乱

　　类似 san 值判定，x >= 0 判定为「守序」，x < 0 判定为「混乱」。
　　初始值为 0，每日经过固定 -1。
　　回归 Day 1 / Day 13 时初始化为 0。

n - 周目判定

　　初始值为 0，达成任意结局之后，如 n >= 0 则 n + 1，同时导入部分「？？」对玩家的称呼变更为 「R-(30+n)」
　　达成 Ending 9 - 绝对理性 后设置 n = -1，开启 EP 4 与隐藏结局
